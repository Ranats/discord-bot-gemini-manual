Gemini API取得手順

この Bot で Gemini を使う場合に必要な API キーの取得手順です。Safari でそのまま開けます。

1. Google AI Studio に入る

実行場所: Safari または Chrome などのブラウザ
  1. ブラウザで https://aistudio.google.com/ を開きます。
  2. 右上の Get started から Google アカウントでログインします。
  3. 利用規約が出たら同意して進みます。

2. プロジェクトを確認する

実行場所: ブラウザ
  1. 左メニューの Dashboard を開きます。
  2. Default Gemini Project が見えていれば、そのまま使って大丈夫です。
  3. もし見当たらなければ新しいプロジェクトを作成します。

3. APIキーを作る

実行場所: ブラウザ
  1. https://aistudio.google.com/projects を開きます。
  2. 中央または上部にある 1キー などの表示をクリックするか、APIキー の画面を開きます。
  3. Create API key のようなボタンがあれば押します。
  4. 表示された API キーをコピーします。
大事です
API キーはパスワードのようなものです。チャットに貼らず、.env にだけ入れてください。

4. Bot に設定する

実行場所: Finder と テキストエディット

.env を開いて、次のように入れます。

LLM_PROVIDER=gemini
GEMINI_API_KEY=ここにコピーしたAPIキー

保存したら、Bot を再起動してください。

5. うまくいかない時

実行場所: ブラウザ / テキストエディット / Finder
  • Google アカウントで AI Studio にログインできているか確認する
  • API キーを最後までコピーできているか確認する
  • .env の GEMINI_API_KEY= の右側に空白や改行が変に入っていないか確認する
  • 保存後に Bot を再起動したか確認する
